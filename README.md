# Gojūon Master - 50音日文學習助手 🇯🇵

一個現代化、精美且功能齊全的日文 50 音學習助手網頁應用。專為初學者設計，結合 Anki 單字卡記憶邏輯、筆順動畫演練與標準語音播放，並針對手機瀏覽器與虛擬鍵盤進行了深度優化，提供最流暢的手機與電腦練習體驗！

---

## ✨ 核心特色

- 🃏 **Anki 記憶卡模式**
  - 輸入正確的羅馬拼音並按 Enter 確認，系統將自動判斷對錯。
  - 對錯回饋會顯示精確的反應時間，並在背面展示平假名與片假名對照。
- 🔁 **智慧間隔重複 (Spaced Repetition)**
  - 答錯的假名會被加入「待複習錯題區」，直到您連續答對兩次將其掌握為止。
  - 支援「啟動錯誤專屬複習」模式，讓您針對弱點集中突破。
- ✍️ **動態筆順演示**
  - 自動載入假名筆順 SVG 動畫（支援自訂 0.5x ~ 3.0x 播放速度，預設 `1.5x`）。
  - 可開啟「手寫繪圖面板」，直接在字卡上進行描紅書寫練習。
- 🔊 **標準語音發音與雙重保障**
  - 支援「答對/答錯時自動發音」與語速調整。
  - **自動線上語音備份 (Online TTS Fallback)**：當使用者電腦系統沒有安裝日文語音包時，程式會自動無縫切換至線上 Google 翻譯發音 API，保證在任何設備上都能 100% 正常播放標準日語發音。
- 📱 **流暢的手機體驗 (Mobile Ready)**
  - 針對手機虛擬鍵盤彈出進行了深度優化，點擊輸入框時會自動將學習區置頂並微縮卡片，確保「題目」與「輸入框」在任何手機上皆能同時完整呈現不重疊。
- 🎨 **現代化毛玻璃介面 (Glassmorphic UI)**
  - 支援高質感的暗黑模式 (Dark Mode) 與明亮模式 (Light Mode) 輕鬆切換。
- 📊 **進度追蹤與假名全表**
  - 提供即時的答題正確率、已掌握假名數量以及學習進度條。
  - 隨時可以開啟「50 音全表」彈窗，點擊即可發音，方便快速查閱。

## 🛠️ 技術棧

- **前端核心**：純原生 HTML5, CSS3, JavaScript (ES6) — 無需安裝任何套件，隨開即用！
- **字型與圖標**：Google Fonts (Outfit & Noto Sans JP), Lucide / Custom SVG Icons
- **外部 API / 資源**：
  - [KanjiVG](https://kanjivg.tagaini.net/) - 提供高品質的假名筆順 SVG 向量線條。
  - Google Translate TTS API - 用作無本地語音包時的發音備份方案。

```text
├── index.html       # 主網頁結構，包含設定欄、字卡區、進度看板與 50音對照彈窗

```

## 📖 如何在本地運行

本專案完全由靜態檔案組成，您不需要配置 Node.js 環境，直接開啟 `index.html` 即可運行！

### 方式 1：直接雙擊打開
在專案目錄下，直接雙擊 `index.html` 即可在瀏覽器中開始學習。

### 方式 2：使用本地 Web 伺服器（推薦，可正常載入筆順與發音）
如果您使用 VS Code，可以安裝 **Live Server** 擴充套件，或在專案根目錄下使用 Python 啟動伺服器：
```bash
# Python 3
python -m http.server 8000
```
然後在瀏覽器中打開 `http://localhost:8000` 即可。
